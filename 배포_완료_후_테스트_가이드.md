### 2단계: 기능 테스트

#### ✅ 테스트 1: 뉴스 검색 기능

1. **키워드 입력**
   - 예: "인공지능", "AI", "기술" 등
2. **"검색" 버튼 클릭**
3. **확인 사항:**
   - 뉴스 10개가 표시되는지 확인
   - 각 뉴스에 제목, 링크, 요약이 있는지 확인
   - 로딩 중 메시지가 표시되는지 확인

**예상 결과:**
- ✅ 뉴스 10개가 왼쪽 열에 표시됨
- ✅ 각 뉴스 항목 클릭 시 원본 링크로 이동
- ✅ 오류 메시지 없음

---

#### ✅ 테스트 2: 검색 기록 기능

1. **"검색 기록" 버튼 클릭**
2. **확인 사항:**
   - 이전에 검색한 키워드들이 표시되는지 확인
   - 검색 시간과 뉴스 개수가 표시되는지 확인
3. **검색 기록 클릭**
   - 이전 검색 내역을 클릭하여 다시 불러오기 테스트

**예상 결과:**
- ✅ 검색 기록 목록이 표시됨
- ✅ 검색 기록 클릭 시 해당 뉴스가 다시 불러와짐
- ✅ 요약도 함께 표시됨

---

#### ✅ 테스트 3: 뉴스 요약 기능

1. **뉴스 검색 후 자동 요약 확인**
2. **확인 사항:**
   - 오른쪽 열에 "뉴스 요약" 섹션이 표시되는지 확인
   - 요약이 150-200자 이내인지 확인
   - 요약 내용이 의미 있는지 확인

**예상 결과:**
- ✅ 검색 후 자동으로 요약 생성됨
- ✅ 요약이 오른쪽 열에 표시됨
- ✅ 요약 길이가 적절함

---

#### ✅ 테스트 4: 챗봇 기능

1. **뉴스 검색 후 챗봇 사용**
2. **질문 입력:**
   - 예: "이 뉴스들의 주요 내용을 알려줘"
   - 예: "가장 중요한 뉴스는 무엇인가요?"
3. **확인 사항:**
   - AI 응답이 정상적으로 나오는지 확인
   - 응답이 검색된 뉴스와 관련이 있는지 확인
   - 대화가 자연스럽게 이어지는지 확인

**예상 결과:**
- ✅ 챗봇에 질문 입력 가능
- ✅ AI 응답이 정상적으로 표시됨
- ✅ 응답이 뉴스 내용과 관련 있음

---

### 3단계: 오류 확인

**브라우저 콘솔 확인:**
1. **F12 키 누르기** (개발자 도구 열기)
2. **Console 탭 확인**
3. **오류 메시지 확인:**
   - 빨간색 오류가 있는지 확인
   - 경고 메시지 확인

**일반적인 오류:**
- ❌ "Failed to fetch" → API 연결 문제
- ❌ "Database connection failed" → 데이터베이스 연결 문제
- ❌ "API key not found" → 환경변수 문제

---

## 🔧 문제 해결

### 뉴스 검색이 작동하지 않아요

**확인 사항:**
1. 브라우저 콘솔에서 오류 확인 (F12)
2. 네트워크 탭에서 API 요청 확인
3. Vercel 배포 로그 확인

**해결 방법:**
- 환경변수 `GEMINI_API_KEY` 확인
- API 요청이 실패하는지 확인

---

### 검색 기록이 표시되지 않아요

**확인 사항:**
1. 뉴스를 먼저 검색했는지 확인
2. 데이터베이스 연결 확인
3. 브라우저 콘솔 오류 확인

**해결 방법:**
- 환경변수 `POSTGRES_PRISMA_URL` 확인
- Vercel 배포 로그에서 데이터베이스 연결 오류 확인

---

### 챗봇이 응답하지 않아요

**확인 사항:**
1. 뉴스를 먼저 검색했는지 확인
2. `GEMINI_API_KEY` 환경변수 확인
3. 브라우저 콘솔 오류 확인

**해결 방법:**
- 환경변수 재확인
- API 요청이 정상적으로 전송되는지 확인
